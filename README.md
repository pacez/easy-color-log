# easy-color-log

Console color grading logs

```javascript
import log from 'easy-color-log';

// or const log = require('easy-color-log').default;

// print - white
log.print(msg);

// success - green
log.success(msg);

// error - red
log.error(msg);

// info - blue
log.info(msg);

// warn - yellow
log.warn(msg);

```

Multiple messages can be passed in.
